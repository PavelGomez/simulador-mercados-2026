Qué es esta plataforma

Esta versión web permite trabajar el modelo de demanda y oferta con funciones lineales, calcular automáticamente el equilibrio y comparar escenarios antes y después de un shock. Está diseñada para apoyar el aprendizaje analítico del curso: no basta con decir “sube la demanda” o “baja la oferta”; se espera una explicación ordenada del mecanismo económico.

Discusión causal esperada: evento → variable → curva → equilibrio.
  • Evento: ¿qué pasó en la realidad o en el caso?
  • Variable: ¿qué variable exógena del modelo cambió?
  • Curva: ¿qué curva se desplaza y en qué dirección?
  • Equilibrio: ¿qué ocurre con el precio y la cantidad de equilibrio?

Ejemplo breve ilustrativo

Evento: sube el ingreso de los consumidores.
Variable: aumenta Y.
Curva: si el bien es normal, la demanda se desplaza a la derecha.
Equilibrio: si la oferta no cambia, tienden a subir P* y Q*.

La meta no es repetir la frase mecánicamente, sino conectar el hecho observado con la variable económica relevante y con el resultado en el equilibrio.

Parámetros de demanda

  • α₀: intercepto.
  • α₁: coeficiente del precio propio.
  • α₂ con Y: ingreso.
  • α₃ con Ps: precio del sustituto.
  • α₄ con Pc: precio del complemento.
  • α₅ con Exp: expectativas.

Parámetros de oferta

  • β₀: intercepto.
  • β₁: coeficiente del precio propio.
  • β₂ con Pi: precio del insumo.
  • β₃ con Tec: tecnología.
  • β₄ con N: número de oferentes.

Lectura de colores

  • Campos amarillos: entradas editables.
  • Resultados rosados: salidas calculadas.
  • Gráficos: muestran solo el primer cuadrante, de modo que las curvas parten desde los ejes.
  • Exp: usar -1, 0 o 1.

Instrucciones precisas de uso

1
Abrir el archivo. Descargue el HTML y ábralo con doble clic o arrástrelo a Chrome, Edge, Safari o Firefox. No requiere internet una vez descargado.
2
Trabajar con campos editables. Modifique solo las casillas amarillas. Los resultados y gráficos se actualizan automáticamente al escribir.
3
Primero use “Simulador”. Sirve para entender cómo un conjunto de parámetros y variables determina una única demanda, una única oferta y un único equilibrio.
4
Luego use “Comparador”. Ingrese un escenario base y después los valores post-shock. La plataforma mostrará el antes, el después y los cambios en ΔP y ΔQ.
5
Explique el mecanismo. La respuesta esperada debe seguir la cadena evento → variable → curva → equilibrio. No basta con reportar números o decir “sube/baja”.
6
Registrar resultados. Para tareas o discusión en clase, anote las ecuaciones y los valores de P* y Q*, o tome una captura de pantalla del escenario trabajado.
Qué se espera de un buen uso
  • Identificar la variable exógena que realmente cambió.
  • Distinguir desplazamiento de curva y movimiento a lo largo de la curva.
  • Expresar el resultado con lenguaje económico claro.
  • Comprobar si el resultado numérico coincide con la intuición económica.
Precauciones prácticas
  • Si el navegador no acepta coma decimal, use punto decimal.
  • Si el equilibrio no existe o no es único, aparecerá una advertencia.
  • En Exp solo se recomienda usar -1, 0 o 1.
  • Esta versión no requiere Excel ni instalación adicional.

Guía breve por rol

Para estudiantes

  1. Lea el caso.
  2. Identifique el shock.
  3. Asocie el shock a una variable del modelo.
  4. Use el comparador para verificar su predicción.
  5. Redacte la explicación causal en 3 a 5 líneas.

Para ayudantes o docencia

  1. Use Simulador para demostración inicial.
  2. Pase luego a Comparador para ejercicios de shock.
  3. Pida primero una predicción verbal, y solo después permita verificar con la plataforma.
  4. Use Ejemplo resuelto como estándar mínimo de calidad explicativa.

Ejemplo resuelto

Estándar mínimo de explicación esperada
  1. Equilibrio base: Qd = 120 - P y Qs = -30 + 2P. Se obtiene P* = 50 y Q* = 70.
  2. Evento: una disrupción geopolítica reduce la capacidad efectiva de oferta.
  3. Variable / curva: cae la oferta; la curva de oferta se desplaza a la izquierda.
  4. Nueva oferta: Qs' = -60 + 2P. La demanda no cambia.
  5. Nuevo equilibrio: P* = 60 y Q* = 60.
  6. Lectura económica: sube el precio y cae la cantidad, porque el ajuste viene desde la oferta.
Obsérvese que aquí el cambio relevante es un desplazamiento de oferta; la demanda no se desplaza. Lo que cambia del lado de la demanda es la cantidad demandada a lo largo de la curva.
